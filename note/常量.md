# 常量

- 常量使用**public static final**修饰的成员变量, **必须有初始化值**, 而且**执行的过程中其值不能被改变**
- 常量的作用和好处: 可以用作系统的配置信息, 方便系统的维护, 哦同时也能提高可读性

## 一. 命名规范

英文字母全部**大写**. 多个单词**下划线**连接起来

## 二. 执行原理

- 在编译阶段会进行"宏替换", 把使用常量的地方全部替换成真实的字面值
- 这样做的好处是让使用常量的程序的执行性能更好

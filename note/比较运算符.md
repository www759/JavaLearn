# 比较运算符

```java
package test;

public class test {
    public static void main(String[] args) {
        int a = 10;
        int b = 20;

        boolean flag = a == b;
        System.out.println(flag);

        System.out.println(a == b);
        System.out.println(a >= b);
        System.out.println(a > b);
        System.out.println(a < b);
        System.out.println(a <= b);
        System.out.println(a != b);
        System.out.println(a = b);
    }
}
```

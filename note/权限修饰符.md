# 权限修饰符

- 权限修饰符: 用来控制一个成员能够被访问的范围的

- 可以修饰成员变量, 方法, 构造器, 内部类, 不同权限修饰符修饰的成员能够被访问的范围将收到限制

  

## 一. 分类和具体作用范围

- 四种范围由小到大: private -> 缺省 -> protected -> public

| 修饰符        | 同一个类中 | 同一个包中其他类 | 不同包下的子类 | 不同包下的无关类 |
| ------------- | ---------- | ---------------- | -------------- | ---------------- |
| **private**   | **√**      |                  |                |                  |
| **缺省**      | **√**      | **√**            |                |                  |
| **protected** | **√**      | **√**            | **√**          |                  |
| **public**    | **√**      | **√**            | **√**          | **√**            |
